# Python Foundations Assignment

Solutions for the three-level Python challenge focusing on clean, efficient code.

## Files

- level_1_prime.py - Checks if a number is prime using functional programming with all()
- level_2_fibonacci.py - Generates Fibonacci sequence using a memory-efficient generator
- level_3_calculator.py - Calculator using Python's operator module for operations

## Quick Start

python level_1_prime.py
python level_2_fibonacci.py
python level_3_calculator.py
## Features

- Input validation on all scripts
- Concise, Pythonic implementations
- Error handling for edge cases
